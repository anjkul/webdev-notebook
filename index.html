Commonly Used GIT Commands

Some commonly used GIT commands are as follows:

git init
Initialize a repository.
git clone
Make a local copy of the remote repository.
git add
Add the modified files in the staging area.
git commit
Save the files in the remote repository/branch.
git push
Deploy the modified files on the host server.
git merge
Transfer the changes from one branch to another.
git branch -av
List the branches of that repository.
git pull
Copy the modified files from the remote repository.

Glossary

This card collects terms and their definitions:

HTML
HTML stands for Hyper Text Markup Language.
Tag
HTML tags are the hidden keywords within a web page that define how the browser must format and display the content.
Element
An element is everything from the start tag to the end tag.
Attributes
Attributes provide additional information about elements - they come in name/value pairs like: name="value"
